# STEP 営業資料骨子

> **毎日1STEP、チームが強くなる。**
> 日報 x 目標 x 案件管理を統合し、PDCAサイクルを自然に定着させるマネジメントSaaS

---

## 1. 表紙

- プロダクト名: **STEP**
- タグライン: 「毎日1STEP、チームが強くなる。」
- サブコピー: 日報提出30秒。チーム状況把握5分。マネジメントを"仕組み"で変える。
- ターゲット業界: 人材紹介・人材派遣・メディア企業

---

## 2. 課題提起 — よくある"3つの痛み"

### メンバーの痛み
- 日報作成に毎日20分以上かかる
- テンプレートが統一されておらず何を書けばいいか分からない
- 書いた日報が読まれているか分からない → モチベーション低下

### マネージャーの痛み
- 誰が提出したか分からず、一人ずつ確認で1時間
- メンバーのコンディション変化に気づけない
- 週次の計画・振り返りが形骸化している

### 経営・管理者の痛み
- チーム全体のKPI進捗が可視化されていない
- 月次レポート作成のために手動集計が発生
- ツールが分散（日報はSlack、案件はスプレッドシート、目標は別ツール...）

---

## 3. ソリューション概要 — STEPとは

### ワンプラットフォームで完結する5つの管理機能

| 機能 | 解決する課題 |
|------|-------------|
| **日報管理** | テンプレート化 + プリフィルで提出30秒。リアクション・コメントで双方向コミュニケーション |
| **週次計画** | 計画 → 承認 → 実行 → 振り返りのPDCAサイクルを仕組み化 |
| **目標管理** | 会社 → 部門 → チーム → 個人のOKR/KPIツリー。日報データから自動追跡 |
| **案件管理** | カンバンビューで案件ステージを可視化。ファネル分析で受注率改善 |
| **ナレッジ** | チーム内ベストプラクティスを蓄積。タグ分類 + 全文検索 |

### Before / After

| ペルソナ | Before | After (STEP導入後) |
|----------|--------|-------------------|
| **メンバー** | 日報に20分。テンプレ選んで全部手入力 | 通知タップ → 前回値プリフィル → 変更点だけ修正 → **30秒で提出** → XP獲得 |
| **マネージャー** | 全員分の日報を1つずつ開いて1時間 | ダッシュボード → 未提出者が赤表示 → ワンクリック確認 → **5分で完了** |
| **管理者** | CSV手動集計で月次レポート作成 | ダッシュボード → KPI自動更新 → 部門別比較 → ドリルダウン → **リアルタイム把握** |

## 4. 機能詳細

### 4-1. 日報管理

- **テンプレートビルダー**: ドラッグ&ドロップでカスタムテンプレート作成
  - フィールドタイプ: テキスト、数値、セレクト、日付、評価、ファイル、リンク、セクション、リピーター
  - テンプレート種別: 日報 / 週報 / 週次計画 / チェックイン
  - ロール別テンプレート割当（営業用、CS用など）
- **前回値プリフィル**: 前日の入力内容を自動反映。変更点だけ修正して提出
- **下書き保存 & 編集**: 途中保存可。下書きはいつでも編集可能
- **リアクション & コメント**: 6種類の絵文字リアクション + スレッド型コメント
- **閲覧ポリシー**: マネージャーのみ / チーム全員 / テナント全員 の3段階

### 4-2. 週次計画 & 承認ワークフロー

- **計画作成**: テンプレートベースの週次計画を毎週作成
- **承認フロー**: draft → submitted → approved / rejected
  - 差戻し時はマネージャーのコメント必須
  - 全履歴を承認ログとして保存
- **週次レビュー**: メンバー自己評価(1-5段階) + 振り返り → マネージャーフィードバック
- **実行率自動算出**: 計画に対する進捗を自動計算

### 4-3. 目標管理 (OKR/KPI)

- **4階層ツリー**: 会社目標 → 部門目標 → チーム目標 → 個人目標
- **KPIフィールドマッピング**: 日報テンプレートの数値フィールドと目標を紐付け
- **進捗自動追跡**: 日報データから目標進捗を日次スナップショット
- **乖離アラート**: 計画ペースと実績の差が5%以上で本人+マネージャーに通知
- **目標期間管理**: 月次/四半期/年次の目標期間を設定

### 4-4. 案件管理 (パイプライン)

- **カンバンビュー**: ドラッグ&ドロップでステージ移動
- **リストビュー**: テーブル形式での一覧・フィルタ・ソート
- **カスタムステージ**: テナントごとにパイプラインステージをカスタマイズ
  - プリセット: アプローチ → 求人獲得 → 推薦 → 書類通過 → 面接 → 内定 → 入社 → 入金
- **ステージ遷移履歴**: 全ステージ変更を記録。停滞案件を可視化
- **ファネル分析**: ステージ別通過率をチャートで可視化

### 4-5. ゲーミフィケーション & エンゲージメント

- **XP & レベルシステム**
  - 日報提出: +10 XP / 週次計画: +15 XP / チェックイン: +5 XP
  - ナレッジ投稿: +20 XP / 目標達成: +50 XP
  - Lv1(0) → Lv2(100) → Lv3(500) → Lv4(1,500) → Lv5(5,000)
- **バッジシステム**: 9種類のバッジ。Common / Rare / Epic / Legendary のレアリティ
  - 初投稿、連続提出、月間目標達成、四半期MVPなど
- **ストリーク**: 連続提出日数をヘッダーに常時表示
- **ピアボーナス**: 毎日1ポイント、チームメンバーに感謝メッセージ付きで送信
  - 専用ページで送受信履歴を一覧表示
  - ダッシュボードにピアボーナス通知
- **アクティビティフィード**: サイドバーにチーム活動をリアルタイム表示

### 4-6. ナッジエンジン (自動リマインダー)

- **提出リマインダー**: 17:00に未提出者へ自動通知
- **再リマインダー**: 18:00に再通知
- **モチベーション低下検知**: 評価スコア3以下が3日連続でマネージャーに通知
- **ソーシャルプルーフ**: チーム提出率を表示して提出を促進
- **案件期限アラート**: 期限超過案件を本人+マネージャーに通知
- **乖離アラート**: 目標と進捗のギャップを自動検知

### 4-7. チーム & 1on1管理

- **チーム編成**: チーム作成・メンバー追加・削除
- **1on1管理**: マネージャー向けアジェンダ管理・コンディション推移チャート
- **Calendar連携**: Calendly / Google Calendar の予約URL設定で1on1予約ボタン表示
- **プロフィールカード**: アバター + 連絡先アクション（電話/Slack/1on1予約/メール）

### 4-8. 週刊STEP (自動ダイジェスト)

- 毎週月曜に自動生成
- 前週の提出率ランキング・MVP・バッジ獲得者を掲載
- チーム全体のハイライトを自動集約

### 4-9. ナレッジベース

- 社内ベストプラクティスの蓄積・共有
- タグ分類 + PostgreSQL全文検索
- チーム内での暗黙知を形式知化

## 5. プラットフォーム機能

### マルチテナント & セキュリティ

| 項目 | 内容 |
|------|------|
| **テナント分離** | 全テーブルにRow Level Security (RLS)。テナントIDはJWTから取得（クライアント入力不可） |
| **認証** | メール/パスワード + Google OAuth。カスタムアクセストークンフック |
| **4ロール権限** | Super Admin / Admin / Manager / Member |
| **監査ログ** | 全操作を追記専用テーブルに記録 |
| **入力バリデーション** | 全Server ActionでZodスキーマ検証 |
| **SSRF防止** | Webhook URLのプライベートIP / localhost拒否 |
| **レート制限** | 全APIエンドポイントにIP制限 |

### 管理者機能

- **ユーザー管理**: 招待・ロール変更・無効化
- **テンプレート管理**: テナント独自テンプレートの作成・編集・公開
- **グローバルテンプレート**: Super Admin がプラットフォーム全体にテンプレート配信
- **パイプラインカスタマイズ**: 案件ステージの追加・編集・並び替え
- **テナント設定**: 組織名・閲覧ポリシー・プラン変更
- **データエクスポート**: ユーザー・日報・案件・目標のCSVダウンロード
- **管理者削除**: 日報・週次計画の管理者削除（監査ログ付き）

### 外部連携

| 連携先 | 内容 |
|--------|------|
| **Slack** | Webhook通知 + スラッシュコマンド (`/step`, `/step-status`) |
| **Chatwork** | API連携によるメッセージ通知 |
| **Google Calendar** | カレンダー連携 |
| **Gmail** | メール連携 |
| **Microsoft Teams** | Teams通知 (計画中) |
| **CTI** | 電話システム連携 (計画中) |
| **REST API** | Bearer / API Key認証の公開API |
| **Webhook送信** | HMAC-SHA256署名付きイベント通知 (11種類のイベント) |

### モバイル & UX

- **PWA対応**: ホーム画面追加・Service Worker
- **モバイルファースト**: レスポンシブデザイン + ボトムナビ + FABボタン
- **コマンドパレット**: `Cmd+K` でグローバル検索・ページ遷移
- **オンボーディングウィザード**: テナント初回セットアップを5ステップでガイド
- **ホワイトラベル**: テナント別カラー・ロゴ・アプリ名カスタマイズ

## 11. よくある質問 (FAQ)

**Q. 導入にIT部門の協力は必要ですか？**
A. 不要です。管理者がブラウザからアカウント作成・テンプレート設定・メンバー招待まで完結できます。

**Q. スマートフォンから利用できますか？**
A. はい。PWA対応でホーム画面に追加するとネイティブアプリのように利用できます。モバイルファーストで設計されています。

**Q. データのセキュリティは大丈夫ですか？**
A. Row Level Securityによるテナント完全分離、全操作の監査ログ、Zodによる入力検証など、エンタープライズグレードのセキュリティを実装しています。

**Q. 既存ツールとの連携は？**
A. Slack、Chatwork連携に対応済み。Webhook送信(11種類)とREST APIで既存システムとの連携も可能です。

**Q. 無料プランから有料プランへの移行は？**
A. データはそのまま引き継がれます。プラン変更は管理画面からいつでも可能です。

**Q. 契約期間の縛りはありますか？**
A. 月額課金で、いつでも解約可能です。年額プランでの割引もご用意しています。
